# STAT437_Final_Project
This code performs original analyses on the Breast Cancer Wisconsin dataset from OncoDatasets based on the methodology used by Pellegrino et al in 'Machine learning random forest for predicting oncosomatic variant NGS analysis (https://doi.org/10.1038/s41598-021-01253-y). We were able to perform PCA and entropy analysis for feature selection, construct LDA and RF models, tune and validate the RF model, and lastly, interpret feature importance. This repo contains individual draft codes, supplemental figures for our text portion, and the assembled project code, titled project_code.Rmd.
